### Для работы бота необходимо установить и настроить следующие компоненты:

### 1. Установка PostgreSQL
- Скачать и установить PostgreSQL с официального сайта
- Во время установки запомнить введённые данные (логин по дефолту, а пароль 1234567890)

### 2. Настройка PostgreSQL
- Запустите pgAdmin 4 (входит в состав PostgreSQL)
- Зарегистрируйтесь в системе при первом запуске
- Подключитесь к серверу PostgreSQL

### 3. Конфигурация подключения

Пользователь: postgres
Пароль: 1234567890
Хост: localhost

### 4. Настройка роли пользователя
1. В pgAdmin 4 слева раздел Servers
2. Перейти в Login/Group Roles
3. Пользователь postgres
4. Правая кнопка мыши → Properties → Definition
5. Пароль: 1234567890

## Установка зависимостей
Перед запуском установите необходимые библиотеки:
telebot psycopg2 uuid вроде

## Запуск проекта
1. Проверить, что PostgreSQL запущен, в случае неконнекта, перезапустить прогу
2. Запустить CreatorDB.py чтобы создать все таблицы
3. Запустить Photos.py чтобы добавить картинки на товары
4. Запусть все три файла бота
